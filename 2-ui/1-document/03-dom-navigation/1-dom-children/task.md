# Дочерние элементы в DOM

Для страницы:

```html
<html>
<body>
  <div>Пользователи:</div>
  <ul>
    <li>Джон</li>
    <li>Пит</li>
  </ul>
</body>
</html>
```

Как получить:
- Напишите код, который получит элемент `<div>`?
- Напишите код, который получит `<ul>`?
- Напишите код, который получит второй `<li>` (с именем Пит)?
