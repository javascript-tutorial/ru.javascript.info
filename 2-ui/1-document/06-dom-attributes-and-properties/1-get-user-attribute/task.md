# Получите атрибут

Напишите код для выбора элемента с атрибутом `data-widget-name` из документа и прочитайте его значение.

```html run
<!DOCTYPE html>
<html>
<body>

  <div data-widget-name="menu">Choose the genre</div>

  <script>
    /* your code */
  </script>
</body>
</html>
```
