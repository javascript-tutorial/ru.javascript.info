# –ù–∞–±–æ—Ä—ã –∏ –¥–∏–∞–ø–∞–∑–æ–Ω—ã [...]

–ù–µ—Å–∫–æ–ª—å–∫–æ —Å–∏–º–≤–æ–ª–æ–≤ –∏–ª–∏ —Å–∏–º–≤–æ–ª—å–Ω—ã—Ö –∫–ª–∞—Å—Å–æ–≤ –≤ –∫–≤–∞–¥—Ä–∞—Ç–Ω—ã—Ö —Å–∫–æ–±–∫–∞—Ö `[‚Ä¶]` –æ–∑–Ω–∞—á–∞—é—Ç "–∏—Å–∫–∞—Ç—å –ª—é–±–æ–π —Å–∏–º–≤–æ–ª –∏–∑ –∑–∞–¥–∞–Ω–Ω—ã—Ö".

## –ù–∞–±–æ—Ä—ã

–î–ª—è –ø—Ä–∏–º–µ—Ä–∞, `pattern:[eao]` –æ–∑–Ω–∞—á–∞–µ—Ç –ª—é–±–æ–π –∏–∑ 3-—Ö —Å–∏–º–≤–æ–ª–æ–≤: `'a'`, `'e'` –∏–ª–∏ `'o'`.

–≠—Ç–æ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è *–Ω–∞–±–æ—Ä–æ–º*.

–ù–∞–±–æ—Ä—ã –º–æ–≥—É—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –≤ —Ä–µ–≥—É–ª—è—Ä–Ω—ã—Ö –≤—ã—Ä–∞–∂–µ–Ω–∏—è—Ö –≤–º–µ—Å—Ç–µ —Å –æ–±—ã—á–Ω—ã–º–∏ —Å–∏–º–≤–æ–ª–∞–º–∏, –Ω–∞–ø—Ä–∏–º–µ—Ä:

```js run

alert( "–¢–æ–ø —Ö–æ–ø".match(/[—Ç—Ö]–æ–ø/gi) ); // "—Ç–æ–ø", "—Ö–æ–ø"
```

–û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –≤ –Ω–∞–±–æ—Ä–µ –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å–∏–º–≤–æ–ª–æ–≤, –Ω–æ –≤ —Ä–µ–∑—É–ª—å—Ç–∞—Ç–µ –æ–Ω —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç —Ä–æ–≤–Ω–æ –æ–¥–Ω–æ–º—É —Å–∏–º–≤–æ–ª—É.

–¢–∞–∫ —á—Ç–æ —ç—Ç–æ—Ç –ø—Ä–∏–º–µ—Ä –Ω–µ –¥–∞—ë—Ç —Å–æ–≤–ø–∞–¥–µ–Ω–∏–π:

```js run

// –∏—â–µ—Ç "–í", –∑–∞—Ç–µ–º [—É –∏–ª–∏ –∞], –ø–æ—Ç–æ–º "–ª—è"
// –∞ –≤ —Å—Ç—Ä–æ–∫–µ –í, –ø–æ—Ç–æ–º —É, –ø–æ—Ç–æ–º –∞
```

–®–∞–±–ª–æ–Ω –∏—â–µ—Ç:

- `pattern:–í`,
- –∑–∞—Ç–µ–º *–æ–¥–∏–Ω* –∏–∑ —Å–∏–º–≤–æ–ª–æ–≤ `pattern:[—É–∞]`,
- –ø–æ—Ç–æ–º `pattern:–ª—è`.

–í —ç—Ç–æ–º —Å–ª—É—á–∞–µ —Å–æ–≤–ø–∞–¥–µ–Ω–∏—è–º–∏ –º–æ–≥—É—Ç –±—ã—Ç—å `match:–í—É–ª—è` –∏–ª–∏ `match:–í–∞–ª—è`.

## –î–∏–∞–ø–∞–∑–æ–Ω—ã

–ï—â—ë –∫–≤–∞–¥—Ä–∞—Ç–Ω—ã–µ —Å–∫–æ–±–∫–∏ –º–æ–≥—É—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å *–¥–∏–∞–ø–∞–∑–æ–Ω—ã —Å–∏–º–≤–æ–ª–æ–≤*.

–ö –ø—Ä–∏–º–µ—Ä—É, `pattern:[a-z]` —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç —Å–∏–º–≤–æ–ª—É –≤ –¥–∏–∞–ø–∞–∑–æ–Ω–µ –æ—Ç `a` –¥–æ `z`, –∏–ª–∏ `pattern:[0-5]` -- —Ü–∏—Ñ—Ä–∞ –æ—Ç `0` –¥–æ `5`.

–í –ø—Ä–∏–≤–µ–¥—ë–Ω–Ω–æ–º –Ω–∏–∂–µ –ø—Ä–∏–º–µ—Ä–µ –º—ã –∏—â–µ–º `"x"`, –∑–∞ –∫–æ—Ç–æ—Ä—ã–º —Å–ª–µ–¥—É—é—Ç –¥–≤–µ —Ü–∏—Ñ—Ä—ã –∏–ª–∏ –±—É–∫–≤—ã –æ—Ç `A` –¥–æ `F`:

```js run

```

–ó–¥–µ—Å—å –≤ `pattern:[0-9A-F]` —Å—Ä–∞–∑—É –¥–≤–∞ –¥–∏–∞–ø–∞–∑–æ–Ω–∞: –∏—â–µ—Ç—Å—è —Å–∏–º–≤–æ–ª, –∫–æ—Ç–æ—Ä—ã–π –ª–∏–±–æ —Ü–∏—Ñ—Ä–∞ –æ—Ç `0` –¥–æ `9`, –ª–∏–±–æ –±—É–∫–≤–∞ –æ—Ç `A` –¥–æ `F`.

–ï—Å–ª–∏ –º—ã —Ö–æ—Ç–∏–º –Ω–∞–π—Ç–∏ –±—É–∫–≤—ã –∏ –≤ –≤–µ—Ä—Ö–Ω–µ–º –∏ –≤ –Ω–∏–∂–Ω–µ–º —Ä–µ–≥–∏—Å—Ç—Ä–µ, —Ç–æ –º—ã –º–æ–∂–µ–º –¥–æ–±–∞–≤–∏—Ç—å –µ—â—ë –¥–∏–∞–ø–∞–∑–æ–Ω `a-f`: `pattern:[0-9A-Fa-f]`. –ò–ª–∏ –ø–æ—Å—Ç–∞–≤–∏—Ç—å —É —Ä–µ–≥—É–ª—è—Ä–Ω–æ–≥–æ –≤—ã—Ä–∞–∂–µ–Ω–∏—è —Ñ–ª–∞–≥ `pattern:i`.

–¢–∞–∫–∂–µ –º—ã –º–æ–∂–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å–∏–º–≤–æ–ª—å–Ω—ã–µ –∫–ª–∞—Å—Å—ã –≤–Ω—É—Ç—Ä–∏ `[‚Ä¶]`.

–ù–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ –º—ã —Ö–æ—Ç–∏–º –Ω–∞–π—Ç–∏ "—Å–∏–º–≤–æ–ª —Å–ª–æ–≤–∞" `pattern:\w` –∏–ª–∏ –¥–µ—Ñ–∏—Å `pattern:-`, —Ç–æ –Ω–∞–±–æ—Ä –±—É–¥–µ—Ç: `pattern:[\w-]`.

–ú–æ–∂–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∏ –Ω–µ—Å–∫–æ–ª—å–∫–æ –∫–ª–∞—Å—Å–æ–≤ –≤–º–µ—Å—Ç–µ, –Ω–∞–ø—Ä–∏–º–µ—Ä `pattern:[\s\d]` –æ–∑–Ω–∞—á–∞–µ—Ç "–ø—Ä–æ–±–µ–ª—å–Ω—ã–π —Å–∏–º–≤–æ–ª –∏–ª–∏ —Ü–∏—Ñ—Ä–∞".

```smart header="–°–∏–º–≤–æ–ª—å–Ω—ã–µ –∫–ª–∞—Å—Å—ã - —Å–æ–∫—Ä–∞—â–µ–Ω–∏—è –¥–ª—è –Ω–∞–±–æ—Ä–æ–≤ —Å–∏–º–≤–æ–ª–æ–≤"
–°–∏–º–≤–æ–ª—å–Ω—ã–µ –∫–ª–∞—Å—Å—ã - –Ω–µ –±–æ–ª–µ–µ —á–µ–º —Å–æ–∫—Ä–∞—â–µ–Ω–∏–µ –¥–ª—è –Ω–∞–±–æ—Ä–æ–≤ —Å–∏–º–≤–æ–ª–æ–≤.

–ù–∞–ø—Ä–∏–º–µ—Ä:

- **\d** -- —Ç–æ –∂–µ —Å–∞–º–æ–µ, —á—Ç–æ –∏ `pattern:[0-9]`,
- **\w** -- —Ç–æ –∂–µ —Å–∞–º–æ–µ, —á—Ç–æ –∏ `pattern:[a-zA-Z0-9_]`,
- **\s** -- —Ç–æ –∂–µ —Å–∞–º–æ–µ, —á—Ç–æ –∏ `pattern:[\t\n\v\f\r ]`, –ø–ª—é—Å –Ω–µ—Å–∫–æ–ª—å–∫–æ —Ä–µ–¥–∫–∏—Ö –ø—Ä–æ–±–µ–ª—å–Ω—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤ –Æ–Ω–∏–∫–æ–¥–∞.
```

### –ü—Ä–∏–º–µ—Ä: –º–Ω–æ–≥–æ—è–∑—ã—á–Ω—ã–π –∞–Ω–∞–ª–æ–≥ \w

–¢–∞–∫ –∫–∞–∫ —Å–∏–º–≤–æ–ª—å–Ω—ã–π –∫–ª–∞—Å—Å `pattern:\w` —è–≤–ª—è–µ—Ç—Å—è –≤—Å–µ–≥–æ –ª–∏—à—å —Å–æ–∫—Ä–∞—â–µ–Ω–∏–µ–º –¥–ª—è `pattern:[a-zA-Z0-9_]`, –æ–Ω –Ω–µ –Ω–∞–π–¥—ë—Ç –∫–∏—Ç–∞–π—Å–∫–∏–µ –∏–µ—Ä–æ–≥–ª–∏—Ñ—ã, –∫–∏—Ä–∏–ª–ª–∏—á–µ—Å–∫–∏–µ –±—É–∫–≤—ã –∏ —Ç.–ø.

–î–∞–≤–∞–π—Ç–µ —Å–¥–µ–ª–∞–µ–º –±–æ–ª–µ–µ —É–Ω–∏–≤–µ—Ä—Å–∞–ª—å–Ω—ã–π —à–∞–±–ª–æ–Ω, –∫–æ—Ç–æ—Ä—ã–π –∏—â–µ—Ç —Å–∏–º–≤–æ–ª—ã, –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–µ –≤ —Å–ª–æ–≤–∞—Ö, –¥–ª—è –ª—é–±–æ–≥–æ —è–∑—ã–∫–∞. –≠—Ç–æ –æ—á–µ–Ω—å –ª–µ–≥–∫–æ —Å –Æ–Ω–∏–∫–æ–¥-—Å–≤–æ–π—Å—Ç–≤–∞–º–∏: `pattern:[\p{Alpha}\p{M}\p{Nd}\p{Pc}\p{Join_C}]`.

–†–∞—Å—à–∏—Ñ—Ä—É–µ–º –µ–≥–æ. –ü–æ –∞–Ω–∞–ª–æ–≥–∏–∏ —Å –∫–ª–∞—Å—Å–æ–º `pattern:\w`, –º—ã –¥–µ–ª–∞–µ–º —Å–≤–æ–π –Ω–∞–±–æ—Ä, –∫–æ—Ç–æ—Ä—ã–π –≤–∫–ª—é—á–∞–µ—Ç –≤ —Å–µ–±—è —Å–∏–º–≤–æ–ª—ã —Å–æ —Å–ª–µ–¥—É—é—â–∏–º–∏ —é–Ω–∏–∫–æ–¥–Ω—ã–º–∏ —Å–≤–æ–π—Å—Ç–≤–∞–º–∏:

- `Alphabetic` (`Alpha`) - –¥–ª—è –±—É–∫–≤,
- `Mark` (`M`) - –¥–ª—è –∞–∫—Ü–µ–Ω—Ç–æ–≤,
- `Decimal_Number` (`Nd`) - –¥–ª—è —Ü–∏—Ñ—Ä,
- `Connector_Punctuation` (`Pc`) - –¥–ª—è —Å–∏–º–≤–æ–ª–∞ –ø–æ–¥—á—ë—Ä–∫–∏–≤–∞–Ω–∏—è `'_'` –∏ –ø–æ–¥–æ–±–Ω—ã—Ö –µ–º—É,
- `Join_Control` (`Join_C`) - –¥–≤–∞ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã—Ö –∫–æ–¥–∞ `200c` –∏ `200d`, –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–µ –≤ –ª–∏–≥–∞—Ç—É—Ä–∞—Ö, –Ω–∞–ø—Ä–∏–º–µ—Ä, –∞—Ä–∞–±—Å–∫–∏—Ö.

–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è:

```js run

let str = `Hi ‰Ω†Â•Ω 12`;

// –Ω–∞–π–¥–µ–Ω—ã –≤—Å–µ –±—É–∫–≤—ã –∏ —Ü–∏—Ñ—Ä—ã

```

–ö–æ–Ω–µ—á–Ω–æ, —ç—Ç–æ—Ç —à–∞–±–ª–æ–Ω –º–æ–∂–Ω–æ –∞–¥–∞–ø—Ç–∏—Ä–æ–≤–∞—Ç—å: –¥–æ–±–∞–≤–∏—Ç—å —é–Ω–∏–∫–æ–¥–Ω—ã–µ —Å–≤–æ–π—Å—Ç–≤–∞ –∏–ª–∏ —É–±—Ä–∞—Ç—å. –ë–æ–ª–µ–µ –ø–æ–¥—Ä–æ–±–Ω–æ –æ –Ω–∏—Ö –±—ã–ª–æ —Ä–∞—Å—Å–∫–∞–∑–∞–Ω–æ –≤ –≥–ª–∞–≤–µ <info:regexp-unicode>.

```warn header="–Æ–Ω–∏–∫–æ–¥–Ω—ã–µ —Å–≤–æ–π—Å—Ç–≤–∞ –Ω–µ —Ä–∞–±–æ—Ç–∞—é—Ç –≤ –Ω–µ–∫–æ—Ç–æ—Ä—ã—Ö —Å—Ç–∞—Ä—ã—Ö –±—Ä–∞—É–∑–µ—Ä–∞—Ö"
–ü–æ–¥–¥–µ—Ä–∂–∫–∞ —é–Ω–∏–∫–æ–¥–Ω—ã—Ö —Å–≤–æ–π—Å—Ç–≤ `pattern:p{‚Ä¶}` –±—ã–ª–∞ –¥–æ–±–∞–≤–ª–µ–Ω–∞ –≤ Edge –∏ Firefox –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –Ω–µ–¥–∞–≤–Ω–æ. –ï—Å–ª–∏ –Ω—É–∂–Ω–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å –ø–æ–¥–¥–µ—Ä–∂–∫—É `pattern:p{‚Ä¶}` –¥–ª—è —É—Å—Ç–∞—Ä–µ–≤—à–∏—Ö –≤–µ—Ä—Å–∏–π —ç—Ç–∏—Ö –±—Ä–∞—É–∑–µ—Ä–æ–≤, –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –±–∏–±–ª–∏–æ—Ç–µ–∫—É [XRegExp](http://xregexp.com/).

–ò–ª–∏ –∂–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–∏–∞–ø–∞–∑–æ–Ω—ã —Å–∏–º–≤–æ–ª–æ–≤ –≤ –∏–Ω—Ç–µ—Ä–µ—Å—É—é—â–µ–º –Ω–∞—Å —è–∑—ã–∫–µ, –Ω–∞–ø—Ä–∏–º–µ—Ä `pattern:[–∞-—è]` –¥–ª—è –∫–∏—Ä–∏–ª–ª–∏—Ü—ã.
```

## –ò—Å–∫–ª—é—á–∞—é—â–∏–µ –¥–∏–∞–ø–∞–∑–æ–Ω—ã

–ü–æ–º–∏–º–æ –æ–±—ã—á–Ω—ã—Ö –¥–∏–∞–ø–∞–∑–æ–Ω–æ–≤, –µ—Å—Ç—å "–∏—Å–∫–ª—é—á–∞—é—â–∏–µ" –¥–∏–∞–ø–∞–∑–æ–Ω—ã, –∫–æ—Ç–æ—Ä—ã–µ –≤—ã–≥–ª—è–¥—è—Ç –∫–∞–∫ `pattern:[^‚Ä¶]`.

–û–Ω–∏ –æ–±–æ–∑–Ω–∞—á–∞—é—Ç—Å—è —Å–∏–º–≤–æ–ª–æ–º –∫–∞—Ä–µ—Ç–∫–∏ `^` –≤ –Ω–∞—á–∞–ª–µ –¥–∏–∞–ø–∞–∑–æ–Ω–∞ –∏ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—Ç –ª—é–±–æ–º—É —Å–∏–º–≤–æ–ª—É *–∑–∞ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ–º –∑–∞–¥–∞–Ω–Ω—ã—Ö*.

–ù–∞–ø—Ä–∏–º–µ—Ä:

- `pattern:[^aeyo]` -- –ª—é–±–æ–π —Å–∏–º–≤–æ–ª, –∑–∞ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ–º `'a'`, `'e'`, `'y'` –∏–ª–∏ `'o'`.
- `pattern:[^0-9]` -- –ª—é–±–æ–π —Å–∏–º–≤–æ–ª, –∑–∞ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ–º —Ü–∏—Ñ—Ä—ã, —Ç–æ –∂–µ, —á—Ç–æ –∏ `pattern:\D`.
- `pattern:[^\s]` -- –ª—é–±–æ–π –Ω–µ–ø—Ä–æ–±–µ–ª—å–Ω—ã–π —Å–∏–º–≤–æ–ª, —Ç–æ –∂–µ, —á—Ç–æ –∏ `pattern:\S`.

–ü—Ä–∏–º–µ—Ä –Ω–∏–∂–µ –∏—â–µ—Ç –ª—é–±—ã–µ —Å–∏–º–≤–æ–ª—ã, –∫—Ä–æ–º–µ –ª–∞—Ç–∏–Ω—Å–∫–∏—Ö –±—É–∫–≤, —Ü–∏—Ñ—Ä –∏ –ø—Ä–æ–±–µ–ª–æ–≤:


## –≠–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–∏–µ –≤–Ω—É—Ç—Ä–∏ [‚Ä¶]

–û–±—ã—á–Ω–æ, –∫–æ–≥–¥–∞ –º—ã —Ö–æ—Ç–∏–º –Ω–∞–π—Ç–∏ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–π —Å–∏–º–≤–æ–ª, –Ω–∞–º –Ω—É–∂–Ω–æ —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞—Ç—å –µ–≥–æ, –Ω–∞–ø—Ä–∏–º–µ—Ä `pattern:\.`. –ê –µ—Å–ª–∏ –Ω–∞–º –Ω—É–∂–Ω–∞ –æ–±—Ä–∞—Ç–Ω–∞—è –∫–æ—Å–∞—è —á–µ—Ä—Ç–∞, —Ç–æ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑—É–µ–º `pattern:\\`, —Ç.–ø.

–í –∫–≤–∞–¥—Ä–∞—Ç–Ω—ã—Ö —Å–∫–æ–±–∫–∞—Ö –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –±–µ–∑ —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–∏—è:

- –°–∏–º–≤–æ–ª—ã `pattern:. + ( )` –Ω–µ –Ω—É–∂–Ω–æ —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞—Ç—å –Ω–∏–∫–æ–≥–¥–∞.
- –¢–∏—Ä–µ `pattern:-` –Ω–µ –Ω–∞–¥–æ —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞—Ç—å –≤ –Ω–∞—á–∞–ª–µ –∏–ª–∏ –≤ –∫–æ–Ω—Ü–µ (–≥–¥–µ –æ–Ω–æ –Ω–µ –∑–∞–¥–∞—ë—Ç –¥–∏–∞–ø–∞–∑–æ–Ω).
